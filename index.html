About DocReadEx

DocReadEx is a free Android app that lets you open, view, and manage documents of all types — plus powerful PDF tools to create and merge PDFs right on your phone.

Most document viewer apps charge for basic features or show constant ads. DocReadEx was built to give everyone a clean, fast, and completely free experience.

Supported File Formats

PDF

Excel (.xlsx)
PowerPoint (.pptx)
Images (JPG/PNG/BMP/GIF)
JSON
YAML
Markdown
Plain Text

PDF Tools

Built-in PDF tools — no separate app needed.

🖼 Images to PDF

Convert any photos or images from your gallery or camera into a PDF document. Choose page size (A4, A3, Letter), orientation, and fit mode.

🔗 Merge PDFs

Combine multiple PDF files into a single document. Pick any PDFs from your device, reorder them, and merge with one tap. Saved directly to Downloads.

Key Features

  • ✔ Smooth PDF viewing with scroll and zoom
  • ✔ View Word, Excel, PowerPoint without Microsoft Office
  • ✔ Image viewer with pinch-to-zoom (JPG, PNG, BMP, GIF)
  • ✔ Convert images to PDF (A4 / A3 / Letter, portrait/landscape)
  • ✔ Merge multiple PDFs into one file
  • ✔ Share documents directly from the app
  • ✔ Recent files — quickly reopen documents
  • ✔ Mark favourite documents for quick access
  • ✔ Open files from other apps (share/intent support)
  • ✔ Clean, simple interface — no clutter
  • ✔ Works fully offline — no internet required
  • ✔ 100% Free — no ads, no subscriptions, no in-app purchases
