About the App

Document Viewer & Scanner is a free Android app that lets you open, view, and manage documents of all types directly on your Android device.

Most document viewer apps charge for basic features or show constant ads. This app was built to give everyone a clean, fast, and completely free experience.

Supported File Formats

PDF
Word (.docx)
Excel (.xlsx)
PowerPoint (.pptx)
Images (JPG/PNG)
JSON
YAML
Markdown
Plain Text

Key Features

  • ✔ Open and view PDF documents with smooth scrolling
  • ✔ View Word, Excel, and PowerPoint without Microsoft Office
  • ✔ Built-in image viewer with pinch-to-zoom
  • ✔ Share documents directly from the app
  • ✔ Mark documents as favourites
  • ✔ Clean, simple interface — no clutter
  • ✔ 100% Free — no in-app purchases, no ads
  • ✔ Works fully offline — no internet required
